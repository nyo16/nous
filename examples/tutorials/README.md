# Tutorials: Structured Learning Path

Learn Nous AI systematically with hands-on tutorials progressing from beginner to expert level.

## Learning Path Overview

### [01-basics/](01-basics/) (15 minutes total)
**Perfect for**: First-time users, getting started

**What you'll learn**:
- Basic agent creation and usage
- Tool calling and function execution
- Provider switching and configuration
- Multi-tool coordination

**Prerequisites**: None

---

### [02-patterns/](02-patterns/) (1 hour total)
**Perfect for**: Users comfortable with basics

**What you'll learn**:
- Real-time streaming responses
- Conversation state management
- Error handling patterns
- ReAct reasoning agents

**Prerequisites**: Complete 01-basics

---

### [03-production/](03-production/) (Advanced level)
**Perfect for**: Building production systems

**What you'll learn**:
- GenServer integration patterns
- Phoenix LiveView integration
- Distributed agent systems
- Monitoring and telemetry

**Prerequisites**: Solid Elixir/Phoenix knowledge, complete 02-patterns

---

### [04-projects/](04-projects/) (Complete applications)
**Perfect for**: Understanding real-world architectures

**What you'll explore**:
- **Council**: Multi-LLM deliberation system
- **Trading Desk**: Enterprise multi-agent coordination
- **Coderex**: AI code editor with specialized tools

**Prerequisites**: Production deployment experience

---

## Tutorial Philosophy

Each tutorial follows this structure:
- **Clear learning objectives** - what you'll achieve
- **Estimated time** - how long it takes
- **Working code** - copy-paste and run
- **Step-by-step explanation** - understand every line
- **Next steps** - clear progression path

All examples are verified working and self-contained.
